# Theme-12: Using STL Sequential Containers

## Description
This laboratory work contains implementations of tasks using STL sequential containers in C++.

## Files
- task1.cpp
- task2.cpp
- task3.cpp

## Implemented tasks
1. Book store management using `vector<Book>`
2. Car list processing using `list<Car>`
3. Delivery order service using `deque<Order>`

## How to run

### Task 1
g++ task1.cpp -o task1
.\task1.exe

### Task 2
g++ task2.cpp -o task2
.\task2.exe

### Task 3
g++ task3.cpp -o task3
.\task3.exe
