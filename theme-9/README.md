# Theme-3: Binary Search Trees in C++

## Description
This laboratory work is dedicated to binary search trees.

## Tasks

### Task 1
- Create BST
- Generate random numbers
- Split into even and odd trees
- Calculate depth

### Task 2
- Read data from file
- Build BST
- Calculate average price
- Count nodes by levels
- Delete subtree

## How to run

### Task 1
g++ task1.cpp -o task1
./task1

### Task 2
g++ task2.cpp -o task2
./task2
