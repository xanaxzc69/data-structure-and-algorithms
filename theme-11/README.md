# Theme-11: Practical Use of Nonlinear Dynamic Structures

## Description
This laboratory work contains implementation of operations on a weighted undirected graph in C++.

## Implemented tasks
- adjacency matrix
- adjacency list
- total number of edges
- vertices with minimum and maximum degree
- check for vertices with equal degree
- search for three vertices with minimum sum of distances
- longest edge of the graph

- task1.cpp

## How to run
g++ task1.cpp -o task1
./task1
